网站生成步骤

1 根据网站需求，由 web-planner 进行创建栏目，并选择合适的模块 参考文件 网站系统模块设计方案.md 选择需要的模块， 并生成内容和栏目 保存为 init-website.json。 然后撰写网站设计风格文档，同时还要生成一个网站中所有需要生成的图片的json 叫做 init-images.json 里面存 文件名称，提示词，Size，Usage和 Alt Text， 提示词用来使用 CogView-3 model via API calls 来生成。

2 开启一个子任务，不要影响主进程的进行。 使用 python脚本（请编写）通过  CogView-3 模型将 init-images.json 的图片 逐一生成。

3 与此同时再 通过python脚本 将init-website.json的内容添加到数据库中。数据库路径 生成网站目录下的 instance\database.db。

4 根据 web-planner 设计的 方案， @website_developer 开始编写模板。

5 网站模板编写后，website_tester 利用 Chrome devtools mcp 对网站进行功能测试，需要测试每个栏目和栏目下的子栏目与链接，看看是否为空或出错，此外还需要检查图片是否缺失。
