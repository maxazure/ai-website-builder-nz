## 🔧 使用说明

### 1. 页面渲染要求
```python
# 在视图中传递以下变量
{
    'page': {
        'title': '页面标题',
        'subtitle': '页面副标题',
        'seo_description': 'SEO描述',
        'keywords': '关键词',
        'hero_media': {
            'path_original': '/path/to/hero-image.jpg'
        },
        'content_html': '<p>页面内容HTML</p>'
    },
    'parent_column': {  # 可选，用于面包屑和侧边栏
        'name': '父栏目名称',
        'slug': 'parent-slug'
    }
}
```

### 2. 图片懒加载使用
```html
<!-- 在内容中使用 data-src 属性 -->
<img data-src="/path/to/image.jpg" alt="图片描述" loading="lazy">
```

### 3. 自定义 CSS 变量
```css
:root {
    --color-primary: #0052cc;
    --color-primary-dark: #003380;
    --color-gray-800: #2d3748;
    --color-gray-900: #1a202c;
}
```
